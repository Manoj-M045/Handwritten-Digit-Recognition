# Handwritten Digit Recognition using Machine Learning

## Overview

This project is a Computer Vision application that recognizes handwritten digits using the Scikit-Learn Digits dataset and the K-Nearest Neighbors (KNN) algorithm. The model is trained on digit images and predicts the corresponding digit accurately.

## Features

* Loads the handwritten digits dataset
* Trains a KNN classification model
* Predicts handwritten digits (0–9)
* Displays the digit image along with prediction results
* Achieves high prediction accuracy

## Technologies Used

* Python
* Scikit-Learn
* Matplotlib
* Machine Learning
* Computer Vision

## Dataset

The project uses the built-in Digits dataset from Scikit-Learn, which contains 1,797 handwritten digit images of size 8×8 pixels.

## Model

* Algorithm: K-Nearest Neighbors (KNN)
* Train-Test Split: 80% Training, 20% Testing
* Accuracy: Approximately 98%

## Conclusion

This project demonstrates the use of Machine Learning and Computer Vision for image classification. The KNN model successfully recognizes handwritten digits with high accuracy and provides a practical introduction to image-based prediction systems.
